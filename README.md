# Relational Databases 1
This is a console for Data Analysis with Python that allows the user to generate code for basic reading and saving data from relational databases.

## Key feature
Single-click buttons for instantly creating syntactically correct code.

## Usage instructions
Click the desired button, and if any user input is required, an alert will indicate where to find the input element.
